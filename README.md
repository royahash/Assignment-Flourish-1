# Assignment-Flourish-1
Flourish Visualization Assignment
This visualization is a dot-density chart that compares road surface temperature measurements across different Road Weather Information Stations (RWIS). Each row represents a station, and each dot corresponds to a fixed unit (1,000) of the total temperature measurement. The dots are stacked in rows, so longer rows indicate higher overall values, making it easy to visually compare stations. Each station is also color-coded, and the exact total values are labeled at the end of each row for precision.

The data comes from the Seattle Department of Transportation’s Road Weather Information Stations dataset, which collects real-time and historical data on road and environmental conditions (such as temperature). The dataset used in this visualization was last updated on April 15th, 2026, as noted in the source at the bottom of the chart.

Link to published visualization: https://public.flourish.studio/visualisation/28548571/
